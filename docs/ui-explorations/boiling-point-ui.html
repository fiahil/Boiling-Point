Why this shape

  1. Volatility is the number that can kill the table, so it's the loudest mark — a big ember numeral, not a buried v2.
  2. Color reads at a glance from the metal frame + the gem, never from a hue alone (the alchemical sigil doubles the signal for color-blind play).
  3. Points sit as warm coins along the foot — secondary, countable.
  4. Effect is a sigil medallion with a name; the full rules live in the hover tooltip so the face stays clean.
Compare today's TUI token: [💙 v2 p1 ◆] — same data, one glyph for "has an effect," no room to explain what.

The table — mid-wave
